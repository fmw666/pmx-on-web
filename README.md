# PMX on Web

本项目是一个基于 WebGL 的 PMX 渲染器，可以在浏览器中加载 PMX 模型并进行渲染。

> 本项目模型来自 [原神官方提供的素材](https://www.aplaybox.com/u/680828836/model)，使用时请遵守其相关协议。
